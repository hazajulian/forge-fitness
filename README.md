# 🏋️ Forge Fitness

A modern and responsive landing page for a fictional premium gym, built with React and Vite.

This project was created as a front-end practice to simulate a real commercial website, focusing on UI/UX, responsive design and reusable components.

> **Note:** Forge Fitness is a fictional brand created exclusively for educational purposes and portfolio demonstration.

---

## ✨ Features

- Responsive design
- Modern landing page
- Hero section with call-to-action
- Benefits section
- About section
- Services overview
- Membership plans
- Trainers section
- Schedule information
- Contact form
- Floating WhatsApp button
- Professional footer
- SEO ready
- Sitemap and robots configuration

---

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/hazajulian/forge-fitness.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
