# 🏋️ Forge Fitness

Landing page moderna y responsive para un gimnasio premium ficticio, desarrollada con React y Vite.

Este proyecto fue creado como práctica de desarrollo Front-End simulando una página comercial real, poniendo énfasis en diseño, experiencia de usuario y componentes reutilizables.

> **Nota:** Forge Fitness es una marca ficticia creada únicamente con fines educativos y para formar parte de un portfolio profesional.

---

## 🚀 Instalación

Clonar el repositorio:

```bash
git clone https://github.com/hazajulian/forge-fitness.git
```

Instalar dependencias:

```bash
npm install
```

Iniciar el proyecto:

```bash
npm run dev
```

Compilar para producción:

```bash
npm run build
```

Previsualizar la versión final:

```bash
npm run preview
```
